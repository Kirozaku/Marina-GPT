## 1. Clone the Repository

```bash
git clone https://github.com/Kirozaku/Marina-GPT.git
cd Marina-GPT
```

## 2. Install Dependencies

```bash
pip3 install -r requirements.txt
```

If your system requires forced installation:

```bash

```

